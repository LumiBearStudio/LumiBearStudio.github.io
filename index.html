Miller Columns

See the full folder hierarchy at once. Click a folder, its contents appear in the next column. Never lose context.

Multi-Tab + Session Restore

Unlimited tabs with tear-off and duplication. Close the app, reopen — every tab exactly where you left it.

Workspaces

Save named tab layouts and restore them instantly. Switch between "Work" and "Personal" in one click.

🔍

Preview Panel

Press Space for Quick Look. Images, video, audio, code with syntax highlighting, PDF, hex, fonts — all built in.

⌨️

Keyboard-First

30+ shortcuts for users who keep their hands on the keyboard. Type-ahead search, full keyboard navigation.

Remote & Cloud

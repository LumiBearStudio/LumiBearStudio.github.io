📂

Miller Columns

See the full folder hierarchy at once. Click a folder, its contents appear in the next column. Never lose context.

📦

File Shelf NEW

macOS Yoink-style holding panel. Stash files from any folder, navigate to the destination, then move or copy them all at once. Pinned items survive restarts.

📋

Multi-Tab + Session Restore

Unlimited tabs with tear-off, duplication, and middle-click to close. Quit and relaunch — every tab exactly where you left it.

📚

Workspaces

Save named tab layouts. Switch between "Work" and "Personal" in one click with Ctrl+Shift+W.

🔎

Preview Panel

Press Space for Quick Look. Images, video, audio, code with syntax highlighting, PDF, hex dump, fonts — all built in.

⌨️

Keyboard-First

30+ shortcuts, customizable bindings, type-ahead search. Your hands never leave the keyboard.

🎨

10 Themes

Dracula, Tokyo Night, Catppuccin, Gruvbox, Nord, One Dark, Monokai, Solarized, Light, Dark.

🔀

Split View

Dual-pane with independent view modes. Drag files between panes. Each pane gets its own preview.

💻

Git Integration

Modified, Added, Deleted, Untracked — status badges right next to your files. Branch info in the status bar.

🌐

Remote & Cloud

FTP/SFTP with saved credentials. OneDrive, iCloud, Dropbox sync badges. Paste files directly from RDP sessions.

✏️

Batch Rename

Regex, prefix/suffix, sequential numbering. Select multiple files, press F2, rename them all at once.

↩️

Full Undo/Redo

Every file operation is tracked. Ctrl+Z to undo, Ctrl+Y to redo. Configurable history depth.

🗑️

Recycle Bin View

Browse, restore, or permanently delete recycled items — with full Miller Columns support.

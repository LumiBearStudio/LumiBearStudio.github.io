## What is SPAN Finder?

SPAN Finder is a free, open-source file explorer for Windows that brings macOS Finder's **Miller Columns** to the Windows desktop. Built with WinUI 3, it's designed for keyboard-first navigation, deep customization, and pro-grade file management workflows.

## New in v1.3.1.0

- 🆕 **File Shelf** — macOS Yoink-style panel: stash files from any folder, move them all at once to the destination (Ctrl+Shift+S)
- 🆕 **.lnk shortcut support** — Click a shortcut to see its target folder in the next column, or preview the target file inline
- 🆕 **F1 opens the User Guide** — Press F1 anywhere in the app to jump here
- 🆕 **745 unit tests** — Rock-solid stability across all features

## System requirements

| Item | Minimum | Recommended |
|:-----|:--------|:------------|
| OS | Windows 10 1809 (build 17763) | Windows 11 |
| Architecture | x64, ARM64 | x64 |
| RAM | 4 GB | 8 GB |
| Disk | 200 MB | 500 MB |

## Microsoft Store (recommended)

The easiest and safest way to install SPAN Finder.

1. Open **Microsoft Store**
2. Search for **SPAN Finder**
3. Click **Get** (or **Install**)
4. Launch from the Start menu

Updates are delivered automatically.

## GitHub Releases (manual)

Prefer to install manually or want a specific version?

1. Visit [GitHub Releases](https://github.com/LumiBearStudio/SpanFinder/releases)
2. Download the latest `.msix` or `.msixbundle`
3. Double-click to install

> **Note:** You may need to enable "Sideload apps" or install our certificate first. See [Manual installation guide](manual-install/) for details.

## Verify installation

After install, launch SPAN Finder. You should see the main window with Miller Columns view.

Press `F1` at any time to open this guide.

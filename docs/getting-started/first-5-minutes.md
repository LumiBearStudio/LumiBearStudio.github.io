## 1. Navigate with Miller Columns

The default view shows **columns**, not folders-and-back-buttons. Click a folder, and its contents appear in a new column to the right. Click again to drill deeper.

- **Left/Right arrow keys**: Move between columns
- **Up/Down arrow keys**: Move between items in a column
- **Enter**: Open the selected folder

[Learn more about Miller Columns](../features/miller-columns/){: .btn .btn-outline }

## 2. Open a new tab

`Ctrl+T` opens a new tab. `Ctrl+W` closes the current tab. `Ctrl+Tab` cycles through tabs.

[Learn more about Tabs](../features/tabs/){: .btn .btn-outline }

## 3. Try the Shelf

Drag any file or folder to the **right edge** of the window. A panel slides out — that's the **Shelf**. Drag more items in, navigate to a different folder, then click **Move here** or **Copy here** to batch-move everything at once.

[Learn more about Shelf](../features/shelf/){: .btn .btn-outline }

## 4. Quick preview

Select any file and press `Space`. A preview pops up — images, code, PDFs, even videos. Press `Space` again to close.

[Learn more about Preview](../features/preview/){: .btn .btn-outline }

## 5. Customize the look

Open Settings (`Ctrl+,`) → **Appearance**. Try a different theme. Adjust density. Change the icon pack.

[Learn more about Themes](../features/themes/){: .btn .btn-outline }

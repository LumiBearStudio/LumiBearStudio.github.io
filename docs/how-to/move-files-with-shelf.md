# Move files across folders using the Shelf

The Shelf is the fastest way to collect files from multiple locations and drop them all into one destination — without juggling clipboard entries or keeping two windows open.

**Time:** ~2 minutes  
**Skill level:** Beginner

---

## The scenario

You've got project assets scattered across three different folders and you need to gather them all into a single delivery folder. With the Shelf:

1. Browse to each source folder and drop files in
2. Navigate to the destination once
3. Click **Move here**

Done. One undo if anything goes wrong.

---

## Step-by-step

### 1. Open the Shelf

Press <kbd>Ctrl</kbd> + <kbd>B</kbd> to show the Shelf panel on the right edge of the window. If you just installed SPAN Finder, the Shelf is enabled by default — hover the right edge and it expands automatically.

### 2. Browse to your first source folder

Use Miller Columns or the sidebar to navigate to the folder containing your first batch of files.

### 3. Select and drop

Select the files you want (<kbd>Ctrl</kbd> + click for individual picks, <kbd>Shift</kbd> + click for a range, or <kbd>Ctrl</kbd> + <kbd>A</kbd> for all). Drag them onto the Shelf panel. You'll see a count badge appear.

**Tip:** You can also right-click selected files and choose **Add to Shelf**.

### 4. Repeat from other folders

Navigate to your next source folder and drop more files. The Shelf accumulates everything — files from different drives, different tab paths, anywhere.

The Shelf shows a live count badge so you always know how many items are waiting.

### 5. Navigate to the destination folder

Browse to where you want everything to end up. The Shelf stays open as you navigate.

### 6. Move or copy

Hover the Shelf to expand it. Click:
- **Move here** — moves all Shelf items to the current folder (originals deleted)
- **Copy here** — copies all Shelf items to the current folder (originals kept)

SPAN Finder runs the operation as a single undoable unit. Press <kbd>Ctrl</kbd> + <kbd>Z</kbd> to reverse it entirely if needed.

### 7. Clear the Shelf

After a successful move, the Shelf automatically empties. Pinned items stay.

---

## Tips

- **Pin important files** — Right-click a Shelf item and choose **Pin**. Pinned items survive Move and Clear operations, so they'll be there on your next session.
- **Use Split View** — Open the destination in the right pane (<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>E</kbd>), collect files in the left pane, then drag directly from the Shelf into the right pane.
- **50-item limit** — The Shelf holds up to 50 items. For bulk moves beyond that, use a search result + drag-and-drop or the filter bar.
